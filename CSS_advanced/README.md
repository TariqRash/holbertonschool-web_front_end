Advanced CSS
============

This directory contains the Advanced CSS project for Holberton School. Each task builds a small, focused piece of the final Techium homepage styling and is stored in its own file under `styles/`.

## Project structure

- `styles/` - One stylesheet per task (`1-style.css` through `32-style.css`).
- `images/` - Required project images (work, people, articles, logos, favicon).

## Requirements

- All files end with a new line.
- Files start with a short task description comment when required by the checker.
- Styles are built incrementally: each task starts from the previous file.
- Tested in Chrome 78.x.
- HTML/CSS should validate with the W3C validator (except tasks that explicitly skip validation).

## Tasks overview

1. Add smooth scrolling behavior.
2. Apply base text colors and visibility helpers.
3. Convert hard-coded values to CSS variables.
4. Store font families in variables and apply them.
5. Store font sizes in variables and apply base sizes.
6. Store font weights in variables and apply them.
7. Integrate Google Fonts in variables.
8. Store line-heights and apply base line height.
9. Remove default link decoration.
10. Center section headers with variables.
11. Style section taglines with variables.
12. Style section titles with variables.
13. Add pseudo-class states for links.
14. Normalize stylesheet for cross-browser consistency.
15. Apply universal box-sizing.
16. Style the `.container`.
17. Add paddings to sections and footer.
18. Style the navigation menu.
19. Add grid utilities and footer tweaks.
20. Clear floats for `.row`.
21. Simplify column selectors.
22. Add dark theme support via data attributes.
23. Fix dark theme elements (footer address, social links).
24. Add backgrounds and hover state to services cards.
25. Create button styles with variables.
26. Add avatar radius and testimonial cite styling.
27. Style the hero section layout.
28. Fix header/logo positioning.
29. Add nav hover effects.
30. Style work cards and overlays.
31. Add quote decoration to testimonials.
32. Add transitions and transforms.

## Usage

Open `index.html` in the browser and link to the desired stylesheet:

```
<link rel="stylesheet" href="styles/32-style.css">
```
